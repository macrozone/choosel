## Howto install



1. install meteor
`$ curl https://install.meteor.com | sh`

2. clone the repository locally: 
`$ git clone https://github.com/macrozone/choosel`

3. `$ cd choosel/src`

4. execute `$ meteor`

5. goto http://localhost:3000
